# IP 配置管理器

基于 Tauri 的 IP、Hosts、代理配置修改器，具有场景切换功能。

## 功能特性

- ✅ 网络状态显示（WiFi/有线网络、静态/动态IP、IP详情）
- ✅ IP 配置修改（静态IP/DHCP切换）
- ✅ Hosts 文件编辑
- ✅ 代理配置管理
- ✅ 场景切换（保存/加载配置场景）

## 开发环境要求

- Node.js 18+
- Rust 1.70+
- Windows 10/11（需要管理员权限）

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run tauri dev
```

## 构建

```bash
npm run tauri build
```

## 使用说明

### 重要提示

⚠️ **需要管理员权限运行**：修改网络配置、Hosts文件和代理设置需要管理员权限。

### 功能说明

1. **网络状态显示**
   - 显示所有网络适配器（WiFi/有线）
   - 显示IP类型（静态/动态）
   - 显示IP地址、子网掩码、网关、DNS等详细信息

2. **IP配置修改**
   - 点击"配置网络"按钮
   - 选择使用DHCP或静态IP
   - 输入IP地址、子网掩码、网关、DNS等信息

3. **Hosts文件编辑**
   - 点击"编辑Hosts"按钮
   - 编辑Hosts文件内容
   - 保存后立即生效

4. **代理配置**
   - 点击"编辑代理"按钮
   - 启用/禁用代理
   - 设置代理服务器和绕过列表

5. **场景管理**
   - 点击"保存当前场景"保存当前所有配置
   - 在场景列表中点击"应用"切换到该场景
   - 点击"删除"移除不需要的场景

## 注意事项

- 修改网络配置可能需要重启网络适配器才能生效
- Hosts文件修改需要管理员权限
- 场景文件保存在应用目录下的 `scenes` 文件夹中
